06. Security Architecture

Google OAuth 2.0 & Session Isolation

Session-Scoped Auto-Logout

Authentication state is stored strictly in sessionStorage. Closing the browser tab or app window automatically wipes session credentials, enforcing fresh login on every launch.

Wildcard DNS (`nip.io`) Integration

Mobile Wi-Fi access via http://10-177-200-94.nip.io:8082 satisfies Google Cloud Console top-level domain validation, eliminating mobile origin_mismatch errors.
