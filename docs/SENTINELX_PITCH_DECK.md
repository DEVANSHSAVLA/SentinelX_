# 💵 OpenCV Computer Vision Currency Scanner
### High-Precision Banknote Topography & Feature Diagnostics

State-of-the-art computer vision inspection for Indian Rupee notes:

- **Optical Thread Shift**: Measures metallic green-to-blue color shift ($ΔE > 8.0$).
- **Microprint Clarity**: Canny edge density extraction ($Index > 0.02$).
- **Mahatma Gandhi Watermark**: Shadow portrait multi-tone gradient check.
- **Non-Currency Classifier**: Automatically rejects non-banknote photos (phones, desks, documents) with **NO INDIAN BANKNOTE DETECTED** alert.
- **Dynamic Serial OCR**: Extracts unique RBI serial numbers (e.g., `7BC 991024`).
