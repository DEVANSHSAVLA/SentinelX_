# SentinelX Desktop Application User Guide

## Native Desktop Features & Controls

1. **System Tray Background Operation**
   - SentinelX Desktop minimizes to the system tray (taskbar on Windows, menu bar on macOS).
   - Monitors incoming platform events from connected backend services in the background with near-zero CPU overhead.

2. **Native OS Notifications**
   - Triggers native Windows Action Center or macOS Notification Center alerts on high-risk scam calls.

3. **Encrypted Local SQLite Vault**
   - Stores offline incident reports and secure credentials using OS Keychain / Windows Vault.

4. **Multi-Window Management**
   - Open dedicated sub-windows for Section 65B PDF Evidence Inspection and Geospatial Hotspot Maps.
